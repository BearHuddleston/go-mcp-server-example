# Security Policy

## Supported Versions

This project is currently maintained on the latest `main` branch.

## Reporting a Vulnerability

Please do not open public issues for security vulnerabilities.

Use one of the following channels:

1. GitHub Security Advisory (preferred):
   - Go to the repository
   - Open the "Security" tab
   - Click "Report a vulnerability"
2. If advisory reporting is unavailable, open a private maintainer contact thread first.

When reporting, include:

- A clear description of the issue and impact
- Reproduction steps or proof-of-concept
- Affected versions/commit SHA
- Suggested remediation if available

We will acknowledge reports as quickly as possible and coordinate responsible disclosure.
